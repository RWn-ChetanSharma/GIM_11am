Tag vs Element vs Attribute

Tag

which comes in < angular brakets >, we have two types of tags.

Tags in Pair ( Start Tag - End Tag ), h1, h2, p, div, etc

Self closing Tag, for eg. br, hr, input, img , etc

Element

Start Tag - Some Content - End Tag

Attribute

They always written in the start tag, with the help of Attribute we can change the behaviour of any element, they provide some additional information
